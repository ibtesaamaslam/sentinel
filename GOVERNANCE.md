# Sentinel Governance

## Project Lead

The Sentinel project is led by the core team, which is responsible for:
- Setting the overall direction and roadmap
- Reviewing and accepting major contributions
- Managing the release process
- Resolving conflicts and disputes

## Core Contributors

Core contributors are individuals who have made significant contributions to the project. They have:
- Write access to the repository
- Voting rights on project decisions
- Responsibility for reviewing PRs in their area of expertise

## Contributors

Contributors are anyone who has submitted a pull request that has been merged. All contributors are expected to follow the Code of Conduct.

## Decision Making

Major decisions are made by consensus among core contributors. When consensus cannot be reached, the project lead makes the final decision.

## Code Review

All changes must be reviewed by at least one core contributor before being merged.
