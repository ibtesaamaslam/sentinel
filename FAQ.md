# Frequently Asked Questions

## General

### What is Sentinel?
Sentinel is an AI Software Engineering Safety Platform that verifies AI-generated code is safe, correct, secure, maintainable, and production-ready.

### Is Sentinel an AI coding assistant?
No. Sentinel is not an AI coding assistant. It is a guardian/reviewer that supervises code generated by AI coding assistants like Claude Code, Cursor, Gemini CLI, and Codex.

### Does Sentinel require an AI API key?
No. All core functionality works offline with no AI API keys required. AI providers are optional.

## Usage

### How do I scan a repository?
```bash
sentinel scan /path/to/project
```

### How do I watch for changes?
```bash
sentinel monitor /path/to/project
```

### What report formats are supported?
Terminal (colored output), HTML (rich report), JSON (machine-readable), and SARIF 2.1.0 (CI/CD integration).

### How do I customize Sentinel?
Run `sentinel init` to generate a configuration file, then edit `sentinel.config.json`.

## Technical

### What languages are supported?
Sentinel is language-agnostic and works with any text-based programming language.

### Is Sentinel fast?
Sentinel uses parallel scanning with configurable worker threads. For large repositories, scan times are proportional to file count.
